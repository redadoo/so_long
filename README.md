## Usage

To play `so_long` follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/redadoo/Cub3d.git
```

2. enter so_long Folder
```bash
cd so_long/
```

3. compile
```bash
make 
```
4. play
```bash
make play 
```
## how to play

* WASD to move.

* esc to quit.
